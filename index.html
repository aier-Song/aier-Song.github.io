【极简教程】小米笔记本安装-Mac-10-15-系统

PS：支持触控版手势，不支持内置 Wi-Fi，蓝牙（可淘宝购买万能驱动解决）！时间有限，暂只提供方案，工具使用方法和 Mac 镜像，网上资源很多。

工具

  • 小米笔记本（我的是 18 款的 Air，原装硬盘不支持增装了一个三星的硬盘）
  • 16g 以上的 U 盘，Etcher，DiskGenius，Paragon Hard Disk Manager 12 Server，EasyUEFI

步骤

  • 使用 Etcher 制作 Mac 启动盘
  • 使用 DiskGenius 替换启动盘 EFI 分区原有的 CLOVER
  • 使用 Paragon Hard Disk Manager 12 Server 制作 Mac 系统格式的硬盘
  • 使用 EasyUEFI 配置启动项
  • 重启选择 Mac 启动盘
  • 按照一般 Mac 系统安装步骤即可完成

黑马面试宝典复习笔记

设计模式

单例模式 : 私有化构造方法，使用统一方法返回实例

工厂模式 : 根据参数创建一个不同类型实例的类

建造者模式 : 根据参数创建多个不同类型实例的类

适配器模式 : 实现一个接口用来扩展【类:继承类，接口:实现接口，对象:内部增加引用】方法的类

策略模式：一个接口两个实现类，多态调用统一方法，实现不同的功能

观察者模式：一个带有通知方法接口的两个实现类，用另一个接口和实现类统一处理前者实例的增删改的同时调用该实例的通知方法，前者为观察者，后者为操作对象

引用类型

强引用：不回收，内存不够报内存溢出

软应用：内存足够则不回收

弱引用：gc 检测到则回收

虚引用：无效引用，gc 立刻回收

jvm 内存分配

方法区 ：常量池，静态变量

堆内存 ：实例对象，程序分配

栈内存 ：方法，对象引用，基本类型，系统自动分配
